# ShareHub - Global File Sharing Platform

A modern, responsive file sharing platform built with HTML, CSS, and JavaScript. ShareHub allows users to upload, download, and share various types of files globally with a focus on high-impact content.

## 🌟 Features

- **Drag & Drop Upload** - Intuitive file upload interface
- **Multiple File Support** - Images, documents, videos, audio, archives
- **50MB File Limit** - Optimized for free hosting
- **Search & Filter** - Find files by name or category
- **Download Tracking** - Monitor file popularity
- **Share Links** - Direct sharing capabilities
- **Responsive Design** - Works on all devices
- **Live Statistics** - Real-time platform metrics

## 🌍 Content Categories

ShareHub focuses on high-impact content that makes a real difference:

- **Educational Resources** - Free textbooks, study guides, coding tutorials
- **Digital Tools & Utilities** - Open-source software, productivity templates
- **Emergency & Preparedness** - First aid guides, disaster preparation
- **Creative & Cultural** - Public domain books, traditional music, recipes

## 🚀 Quick Start

### Prerequisites
- Git installed on your system
- GitHub account
- Web browser

### Installation

1. **Clone this repository:**
   ```bash
   git clone https://github.com/yourusername/sharehub.git
   cd sharehub
   ```

2. **Open in browser:**
   ```bash
   # Simply open index.html in your preferred browser
   open index.html
   ```

### Deployment

#### Option 1: Vercel (Recommended)
1. Push code to GitHub
2. Connect repository to [Vercel](https://vercel.com)
3. Deploy automatically

#### Option 2: Netlify
1. Push code to GitHub
2. Connect repository to [Netlify](https://netlify.com)
3. Deploy as static site

#### Option 3: GitHub Pages
1. Push code to GitHub
2. Enable GitHub Pages in repository settings
3. Select `main` branch as source

## 🔧 Technical Details

### Current Implementation
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Storage:** Browser localStorage (demo purposes)
- **Styling:** CSS Grid, Flexbox, CSS Variables
- **Icons:** Font Awesome 6.0
- **Fonts:** Google Fonts (Inter)

### For Production
To make this a fully functional file sharing platform, you'll need:

1. **Backend Server:** Node.js/Express or Python/Flask
2. **Database:** PostgreSQL (Supabase) or MongoDB
3. **File Storage:** AWS S3 or Google Cloud Storage
4. **Authentication:** JWT or OAuth
